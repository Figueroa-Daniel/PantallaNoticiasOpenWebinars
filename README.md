# Pantalla de Noticias - OpenWebinars

Este proyecto es una aplicación de ejemplo desarrollada como parte del curso de **OpenWebinars**:
**Desarrollo de Aplicaciones Android: Kotlin 2.0 y Jetpack Compose**.

## Descripción

La aplicación muestra una pantalla de noticias moderna utilizando las últimas tecnologías de desarrollo nativo en Android. Implementa una arquitectura basada en **ViewModel** y **StateFlow**, con una interfaz de usuario construida íntegramente con **Jetpack Compose**.

### Características principales:

*   **Kotlin 2.0**: Uso de las últimas funcionalidades del lenguaje.
*   **Jetpack Compose**: UI declarativa para una experiencia de usuario fluida.
*   **Material Design 3**: Implementación de las guías de diseño actuales de Google.
*   **Gestión de Estados**: Uso de `StateFlow` y estados sellados (`Sealed Classes`) para manejar la carga de datos.
*   **Componentes Personalizados**: Cabeceras dinámicas, información meteorológica y listas optimizadas con `LazyColumn`.
*   **Carga de Imágenes**: Integración con la librería Coil para la gestión de imágenes remotas.
